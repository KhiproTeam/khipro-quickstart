## ক্ষিপ্র কিবোর্ডের বিশেষত্ব
প্রচলিত ফোনেটিক হোক কিংবা ফিক্সড লেআউট হোক, উভয় ধরনের লেআউটের চেয়েই ক্ষিপ্র-তে দ্রুত লেখা যায়। ক্ষিপ্রর প্রধান আকর্ষণ যে ফিচারগুলো, সেগুলো জানতে এখানে ক্লিক করুন।
# ক্ষিপ্রসূচন | *ক্ষিপ্র*-র সহজ টিউটোরিয়াল
## ধাপ ১: সাধারণ বর্ণ লেখা
অধিকাংশ বর্ণকে কাছাকাছি উচ্চারণের বর্ণ দিয়ে লেখা যাবে।\
যেমন:
```mermaid
flowchart TD
A(amar) --> B("আমার")
C(kolom) --> D("কলম")
E("khela") --> F("খেলা")
```
> [!IMPORTANT]
> `আ` লিখতে `a`, এবং, `এ` লিখতে `e` ব্যবহার করা হয়।

> [!NOTE]
> `অ্যা` ধ্বনি লেখার একটি বিশেষ ব্যবস্থা _ক্ষিপ্র_ -তে রয়েছে যেটা পরে উল্লেখ করা হবে।

**নিচের বর্ণগুলোকে হুবহু উচ্চারণ অনুসারে কিংবা, _অভ্র_ -এর মতো মেথডগুলোর অনুরূপ উপায়ে _ক্ষিপ্র_ -তে লেখা যায়:**

|---|------|------|------|------|
| o | a    | i    | u    | e    |

| ক | খ | গ | ঘ | ঙ    | চ | ছ | জ | ঝ |
|---|---|---|---|-------|---|---|---|---|
| k | kh| g | gh| ng | c | ch | j | jh |

| ত | থ | দ | ধ | ন |
|---|---|---|---|---|
| t | th| d | dh| n |

| প | ফ | ব | ভ | ম | য | র | ল | শ |
|---|---|---|---|---|---|---|---|---|

| স | হ | য় |
|---|---|---|

এসব বর্ণ দিয়ে আরো কিছু সহজ ও কমন উদাহরণ:  

```mermaid
flowchart TD
A(nouka) --> B("নৌকা")
c(vitor) --> d("ভিতর")
e(shorbot) --> f("শরবত")
g(chiniye) --> h("ছিনিয়ে")
```

## ধাপ ২: অন্যান্য ব্যঞ্জনবর্ণ লেখা

### ধাপ ২.১: খ, ঘ, ছ, ঝ, ইত্যাদি মহাপ্রাণ বর্ণ লেখা
নিচের সাতটি মহাপ্রাণ বর্ণ লেখার জন্য h ব্যবহার করতে হবে:

```mermaid
flowchart TD
a(kh) --> b(খ)
c(gh) --> d(ঘ)
e(ch) --> f(ছ)
g(jh) --> h(ঝ)
```

```mermaid
flowchart TD
i(th) --> j(থ)
k(dh) --> l(ধ)
m(ph) --> n(ফ)
```

> [!NOTE]
> ভ, শ, ঠ, ঢ, ঢ় ইত্যাদি বর্ণ `h` যোগে লেখা যাবে না। এগুলো লেখা আমরা পরবর্তী ধাপে শিখব।

### ধাপ ২.২: ভ লেখা
```mermaid
flowchart LR
a(v) --> b(ভ)
```

> [!IMPORTANT]
> `ভ` কেবল `v` দিয়ে লেখা যাবে, `bh` দিয়ে নয়। যেমন: `abhawa` = `আবহাওয়া`।\
`ফ` কেবল `ph` দিয়ে লেখা যাবে, `f` দিয়ে নয়। কেননা `f` মডিফায়ার key. 

### ধাপ ২.৩: শ লেখা
`শ` যদিও মহাপ্রাণ বর্ণ নয়, তবুও `sh` দিয়ে `শ` লেখা যাবে।
```mermaid
flowchart LR
a(sh) --> b(শ)
```

## ধাপ ৩: মডিফায়ার
ক্ষিপ্র-তে `f` key কে মডিফায়ার হিসেবে রাখা হয়েছে। কিছু কিছু বর্ণের পরে `f` লিখে সেগুলোকে মডিফাই করা যাবে।\
মডিফায়ার ক্ষিপ্র-র একটি শক্তিশালী ফিচার। সামনে মডিফায়ারের অনেক গুলো উপযোগীতা ও উপকারীতা দেখতে পাবো আমরা।

## ধাপ ২: (বাকি অংশ)
### ধাপ ২.৪: ট, ঠ, ড, ঢ, ণ, ষ, ড়, ঢ়, ইত্যাদি বর্ণ লেখা
* ত -এর পরে মডিফায়ার (`f`) ব্যবহার করে ট, ঠ;
* দ -এর পরে `f` ব্যবহার করে ড, ঢ;
* ন -এর পরে `f` দিয়ে ণ;
* স -এর পরে `f` দিয়ে ষ;
* এবং, র -এর পরে `f` দিয়ে ড়, ঢ় লেখা যাবে।  

একবার অথবা দুইবার `f` ব্যবহার করে লেখা যাবে নিচের মতো:

```mermaid
flowchart TD
a(tf) --> b(ট)
c(tff) --> d(ঠ)
e(df) --> f(ড)
g(dff) --> h(ঢ)
```

```mermaid
flowchart TD
a(nf) --> b(ণ)
c(sf) --> d(ষ)
e(rf) --> f(ড়)
g(rff) --> h(ঢ়)
```

### ধাপ ২.৫: ক্ষ
`ক্ষ` যুক্তবর্ণ (ক্ষ = ক + ষ = `k` + `sf`) হলেও এটা লেখার দুটো শর্টকাট আছে: `kf`, ও `kkh`

```mermaid
flowchart TD
A(kf) --> B(ক্ষ)
c(kkh) --> B(ক্ষ)
e(ksf) --> B(ক্ষ)
```

ক্ষ নিয়ে পূর্ণাঙ্গ আলোচনা আমাদের পূর্ণাঙ্গ ডকুমেন্টেশনে করা হয়েছে। (ক্লিক করুন)


### ধাপ ২.৬: ঞ
`ঞ` লেখার জন্য প্রায় সর্বদাই `n` দিয়েই কাজ চলে যাবে। কেননা ঞ যুক্তবর্ণরূপে ছাড়া দেখা যায় না বললেই চলে।
তবুও মুক্ত রূপে ঞ লেখার জন্য `nff` ম্যাপিং রয়েছে।

```mermaid
flowchart TD
a(nj) --> b(ঞ্জ)
e(nc) --> f(ঞ্চ)
c(nff) --> d(ঞ)
```

### ধাপ ২.৭: ং লেখা
`x` দিয়ে `ং` লিখতে হবে। 
এটি ব্যতিক্রমী হলেও এতে বেশ কিছু শব্দ লিখতে সুবিধা হবে। যেমন:

```mermaid
flowchart TD
a(oxk) --> e(অংক)
b(ongk) --> f(অঙ্ক)
c(lixk) --> g(লিংক)
d(lingk) --> h(লিঙ্ক)
```

অর্থাৎ, এতে `ং` এবং `ঙ` এর মধ্যে পার্থক্য করা সহজ হবে।\
এ বিষয়ে বিস্তারিত পড়ুন আমাদের পূর্ণাঙ্গ ডকুমেন্টেশনে। পড়তে ক্লিক করুন।

## ধাপ ৪: অন্যান্য স্বরবর্ণ লেখা
কিছু স্বরবর্ণ লেখার জন্য ক্ষিপ্রতে ব্যতিক্রম ম্যাপিং ব্যবহার করা হয়। যেমন:

```mermaid
flowchart TD
a(ii) --> d(ঈ)
b(uu) --> e(ঊ)
c(q) --> f(ঋ)
```

```mermaid
flowchart TD
a(w) --> e(ও)
b(wi) --> f(ঐ)
c(wu) --> g(ঔ)
```

> [!NOTE]
> `wi` = `ঐ`, `wu` = `ঔ` নির্ধারণ করায় এসব শব্দ লিখতে সুবিধা হবে: `boi` = `বই`, `bou` = `বউ`

পূর্ণাঙ্গ ডকুমেন্টেশনে এই বিষয়ে বিশদে আলোচনা করা হয়েছে। এ বিষয়ে সবটা জানতে ক্লিক করুন।

## ধাপ ৫: যুক্তবর্ণ লেখা এবং এড়িয়ে যাওয়া
1. যুক্তবর্ণ গঠন সম্ভব এমন একাধিক বর্ণ একসাথে টাইপ করলে যুক্তবর্ণ গঠিত হয়ে যাবে।\
যেমন:

```mermaid
flowchart TD
A(kt) --> B("ক্ত")
c(kr) --> d("ক্র")
```

2. যুক্তবর্ণ এড়াতে বর্ণের মাঝে *পৃথায়ক* ব্যবহার করুন। ক্ষিপ্রতে সেমিকোলন `;` হলো *পৃথায়ক।*\
যেমন:  
```mermaid
flowchart TD
c(k;r) --> d("কর")
e(k;t) --> f("কত")
```

3. পৃথায়কের পরিবর্তে `o` ব্যবহার করেও যুক্তবর্ণ এড়ানো যায়।\
যেমন:
```mermaid
flowchart LR

c(kor) --> d("কর")
e(kot) --> f("কত")
```

> [!NOTE]
> সেমিকোলন `;` যেহেতু *পৃথায়ক*, তাই কোথাও সেমিকোলন লিখতে পরপর দুবার সেমিকোলন চাপুন।

যেমন: `sohoj;;` = `সহজ;`

> [!NOTE]
> প্রশ্ন হতে পারে; যুক্তবর্ণ এড়ানোয় `o` থাকা সত্ত্বেও *পৃথায়ক*-এর ধারণা কেন আনা হয়েছে। এতে কিছু সুবিধা রয়েছে, যা পরবর্তী ধাপগুলোর আলোচনায় উঠে আসবে।

### ধাপ ৫.২: স্ল্যাশ `/` -এর ব্যবহার (যুক্তবর্ণ সংক্রান্ত)
পূর্বে উল্লেখিত মডিফায়ারের মতো `/` key -কেও ক্ষিপ্রতে মডিফায়ার হিসেবে রাখা হয়েছে।

উচ্চারণ অনুসারে লিখতে গিয়ে চলে আসা অবাঞ্ছিত যুক্তবর্ণ দূরীকরণের জন্য স্ল্যাশ ব্যবহার করা যাবে।\
যেমন:\
`বাকরুদ্ধ` লেখার জন্য কেউ `bakruddh` লিখলে `বাক্রুদ্ধ` আসবে। এখানে `ক্র` চলে আসার পরে যদি `/` দিয়ে দেওয়া হয় তাহলে ব্যাকস্পেস না চেপেই সেটাকে ভেঙে ফেলা যাবে। `bakr/uddh` = `বাকরুদ্ধ` উদাহরণ:

```mermaid
flowchart TD
a(alp/ona) --> b(আলপনা)
e(lagb/e) --> f(লাগবে)
z(aelg/oridom) --> g(অ্যালগরিদম)
h(ajh/ari) --> i(আজহারি)
```

এ বিষয়ে পূর্ণাঙ্গ আলোচনা আমাদের পূর্ণাঙ্গ ডকুমেন্টেশনে করা হয়েছে। এ বিষয়ে বাকিটা জানতে ক্লিক করুন।
