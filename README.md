## ধাপ ২: যুক্তবর্ণ লেখা এবং এড়িয়ে যাওয়া
1. যুক্তবর্ণ গঠন সম্ভব এমন একাধিক বর্ণ একসাথে টাইপ করলে যুক্তবর্ণ হয়ে যায়।\
যেমন:

```mermaid
flowchart TD
A(kt) --> B("ক্ত")
c(kr) --> d("ক্র")
```

2. যুক্তবর্ণ এড়াতে বর্ণের মাঝে *পৃথায়ক* ব্যবহার করুন। ক্ষিপ্রতে সেমিকোলন `;` হলো *পৃথায়ক।*\
যেমন:  
```mermaid
flowchart TD
A(k;t) --> B("কত")
c(k;r) --> d("কর")
```

3. পৃথায়কের পরিবর্তে `o` ব্যবহার করেও যুক্তবর্ণ এড়ানো যায়।\
যেমন:
```mermaid
flowchart TD
A(kot) --> B("কত")
c(kor) --> d("কর")
```

> [!NOTE]
> লক্ষ্যণীয়, সেমিকোলন `;` যেহেতু *পৃথায়ক*, তাই কোথাও সেমিকোলন লিখতে পরপর দুবার সেমিকোলন চাপুন।

যেমন:
```mermaid
flowchart TD
A(sohoj;;) --> B("সহজ;")
```

> [!NOTE]
> প্রশ্ন হতে পারে; যুক্তবর্ণ এড়ানোয় `o` থাকা সত্ত্বেও *পৃথায়ক*-এর ধারণা কেন আনা হয়েছে। এতে কিছু সুবিধা রয়েছে, যা পরবর্তী ধাপগুলোর আলোচনায় উঠে আসবে।
