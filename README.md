## ধাপ ১: সাধারণ বর্ণ লেখা
অধিকাংশ বর্ণকে কাছাকাছি উচ্চারণের বর্ণ দিয়ে লেখা যায়।  
যেমন:
<p>

`amar` = `আমার`  
`kolom` = `কলম`  
`khela` = `খেলা` 

</p>

* লক্ষ্যণীয়, `া`কার লিখতে `a`, এবং, `ে`কার লিখতে `e` ব্যবহার করা হয়।
* পরবর্তী ধাপগুলোয় আমরা বিশেষ বিশেষ স্বর, এবং ব্যঞ্জন লেখা শিখব।
